Waitlist Form Test Page

Testing the waitlist form locally

This page allows you to test the waitlist form without setting up a Jekyll site.

  • Make sure npm run start:dev is running in the the-widget directory
  • API endpoint: http://localhost:3000/api/subscribe
  • This page is accessible at: http://localhost:3000/local-test/test-form.html

Basic Form (Default Settings)

Standard form with default API URL and source tracking.

Turnstile CAPTCHA: This test form doesn't include CAPTCHA. To test with Turnstile, see the Jekyll include at jekyll/_includes/waitlist-form.html and add your site key via the turnstile_site_key parameter.
